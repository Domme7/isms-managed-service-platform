# Context Packs

Context Packs sind kuratierte Arbeitskontexte für genau ein Work Package. Sie ersetzen nicht die Konzeptquellen, sondern verweisen auf die relevanten Abschnitte.

Leitplanken:

- Zielgröße 8–24 KB Text
- Ziel, Nicht-Ziel, Acceptance Criteria, Verträge, Security-Regeln und Quellen
- keine duplizierten Volltexte
- Quelldokumentversionen und Hashes nachvollziehbar
- bei Abschluss archivieren oder aktualisieren
