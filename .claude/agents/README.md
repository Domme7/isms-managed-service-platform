# Agentenverträge

Dieses Verzeichnis enthält **26 portable Role Contracts** aus Dokument 20B und Dokument 21.

Nicht alle Agenten werden gleichzeitig aktiviert. Der CEO-/Orchestrator wählt nur Rollen, deren Ergebnis für das aktive Work Package notwendig ist. Builder und Reviewer bleiben getrennt; read-only Reviewrollen erhalten standardmäßig keine Schreib- oder Merge-Rechte.

Die konkrete Frontmatter- und Toolsyntax muss in WP-001 gegen die installierte Claude-Code-Version geprüft werden.
