# Security- und Privacy-Regeln

- Keine Secrets oder realen Kundendaten.
- Least Privilege, Tenant Isolation und nachvollziehbare Autorisierung.
- Logs enthalten keine Tokens, Credentials oder unnötige personenbezogene Daten.
- Kritische Exporte, Löschungen und externe Aktionen benötigen Human Gate.
- Security Findings können Merge/Release blockieren.
