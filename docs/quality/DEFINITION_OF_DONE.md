# Definition of Done

Ein Work Package ist erst abgeschlossen, wenn:

1. Acceptance Criteria erfüllt sind.
2. Outcome demonstrierbar ist.
3. relevante Unit-, Contract-, Integrations-, Security-, Privacy-, Tenant-, Accessibility- und E2E-Tests ausgeführt sind.
4. Dokumentation, Status, Work Queue und Handover aktuell sind.
5. notwendige ADRs/Decision Records existieren.
6. keine ungeklärten kritischen Findings bestehen.
7. unabhängiger Review dokumentiert ist.
8. ein Verified oder Release Checkpoint vorliegt.
9. die nächste Session ohne Chatwissen fortsetzen kann.
